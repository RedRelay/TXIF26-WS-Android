# Un webservice prototype pour l'application de livraison de la TX "Sauvons IF26 - Partie Android"

## Prérequis :
* Node.js
* NPM

## Installation
* Ouvrir Node Command Prompt
* Se placer dans le repertoire courant
* npm install restify
* npm install restify-errors

## Démarrage du webservice
* Ouvrir Node Command Prompt
* Se placer dans le repertoire courant
* node webservice.js
